تقرير مشروع التخرج: متجر Tech Phone Store
المقدمة
يهدف هذا المشروع إلى إنشاء منصة تجارة إلكترونية شاملة لبيع الهواتف الذكية والإكسسوارات. ستركز المنصة على توفير تجربة تسوق آمنة، فعالة، وموثوقة للمستخدمين من خلال واجهات مستخدم متقدمة، وتقنيات حديثة مثل HTML, CSS, JavaScript, PHP، و MySQL.
الفصل الأول: وصف وفكرة المشروع
رؤية المشروع
يهدف المشروع إلى أن يكون وجهة رئيسية للمتسوقين عبر الإنترنت الباحثين عن هواتف ذكية ومستلزماتها.
تحليل المشكلات والحلول
معالجة المشكلات الشائعة مثل تشتت معلومات المنتجات، صعوبة الوصول، وغياب الشفافية بعد الشراء. سيتم تقديم حلول من خلال نظام فهرسة موحد، تجربة شراء مرنة، وتحديثات لحالة الطلبات.
الفصل الثاني: المتطلبات والمنهجية الفنية
المتطلبات الأساسية
النظام يتكون من واجهة أمامية متكاملة مع وظائف إدارة المستخدمين، الطلبات، والدفع، بالإضافة إلى لوحة تحكم إدارية للمسؤولين.
التقنيات المستخدمة
تشمل HTML, CSS, JavaScript, PHP و MySQL لضمان أداء مستقر وسهل التطوير.
الفصل الثالث: تحليل وتجربة المستخدم
قصص المستخدم
تم تحديد متطلبات النظام بناءً على قصص المستخدمين، مثل القدرة على البحث والتصفية، إدارة سلة التسوق، وتتبع حالة الطلبات.
رحلة المستخدم
تم تصميم مسار تجربة المستخدم لضمان سهولة الاستخدام منذ اكتشاف المنتج وحتى إتمام عملية الدفع وتتبع الطلب.
الفصل الرابع: تصميم قاعدة البيانات
مخطط الكيانات والعلاقات
يشمل النظام جداول مختلفة مثل "Users" و "Products" و "Orders" و "OrderItems" بالإضافة إلى العلاقات بين هذه الجداول لتسهيل تنظيم البيانات.
الفصل الخامس: الشاشات الخاصة بالنظام
الشاشات المنجزة
تم تطوير واجهات مختلفة للنظام، تشمل صفحة التسجيل، صفحة المنتج، سلة التسوق، صفحة الدفع، وتتبع الطلبات. هذه الشاشات تهدف إلى تسهيل تجربة المستخدم وضمان تسوق سلس وآمن.
الشاشة 1: الصفحة الرئيسية
تحتوي على عرض للمنتجات البارزة، تصنيفات المنتجات، ونظام البحث.
الوظائف: عرض المنتجات حسب التصنيف، إمكانية البحث باستخدام الفلاتر، عرض تفاصيل المنتج عند النقر عليه.
الشاشة 2: صفحة تسجيل الدخول والتسجيل
تسمح للمستخدم بتسجيل الدخول أو إنشاء حساب جديد.
الوظائف: إدخال بيانات المستخدم مثل الاسم، البريد الإلكتروني، وكلمة المرور. إمكانية استرجاع كلمة المرور في حالة نسيانها.
الشاشة 3: صفحة عرض المنتج
تحتوي على تفاصيل المنتج مثل المواصفات، السعر، والتقييمات.
الوظائف: إضافة المنتج إلى سلة التسوق، عرض صور متعددة للمنتج، عرض المعلومات التقنية مثل المواصفات والعروض.
الشاشة 4: صفحة سلة التسوق
تحتوي على قائمة المنتجات التي تم إضافتها للسلة.
الوظائف: تعديل الكميات أو حذف المنتجات، عرض المجموع الكلي للطلب، الانتقال لصفحة الدفع.
الشاشة 5: صفحة الدفع
تحتوي على إدخال بيانات الشحن والدفع.
الوظائف: إدخال عنوان الشحن، اختيار طريقة الدفع، تأكيد الطلب.
الشاشة 6: صفحة تتبع الطلبات
تحتوي على حالة الطلبات السابقة.
الوظائف: عرض حالة الطلب (قيد المعالجة، تم الشحن، تم التوصيل). إمكانية تتبع الشحنة باستخدام رقم التتبع.
الشاشة 7: لوحة تحكم المسؤول
واجهة لإدارة المنتجات، الطلبات، والمستخدمين.
الوظائف: إضافة، تعديل، أو حذف المنتجات، إدارة الطلبات، عرض التقارير والإحصائيات، إدارة المستخدمين وصلاحياتهم.
